Introduction to C

C is an old-school programming language, developed in 1972 by Dennis Ritchie at bell laboratories of AT&T. It was developed to help overcome some of the problems faced with previous languages and as a result, has come rich with innovative features used in more modern programming languages today.

Some of the main key features of C include:

  • Rich library with many built in functions for its time
  • Memory management for control over your program's RAM usage
  • Pointers for directly interacting with the system's memory
  • Recursion to provide powerful code reusability

Getting Started

C does not require any previous programming knowledge. However, it is advised you are comfortable enough to use text-editors, terminals, and file management on your laptop or PC.

Hello World

To get started with writing your first program, simply type into your text editor the following:

#include <stdio.h>

int main(void) {
    printf("Hello, world!\n");
}

It's worth noting that '\n' will not show up in the terminal. The value \n is actually used as an indicator for C to start a new line.

Variables

There are many different variable available in C. A few of the most common types include:

  • int - Integers
  • char* - Characters (Don't worry about the * just yet)
  • bool - Boolean
  • float - Floating point numbers, or numbers with decimals
  • double - For large integers

To declare a variable you can do the following:

int x = 5;

You denote the variable you want to use, give it a name, and then assign it a value, using the assignment operator '='. This is commonly mistaken as an equals sign, however, in C, it's simply used to assign the value on the right to the value on the left. You then finish each line of code with a ';'

Operators

On top of having different variables we can use in C. There are also a number of operators available that help you perform arithmetic on your variables.

Here's a list of some of the most popular ones:

  • + for addition
  • - for subtraction
  • * for multiplication
  • / for division
  • % for remainder

if else Statements

if else statements are easy ways of being able to check if a condition is true or false. If the statement is true, you perform a set of actions, else, you perform an alternative set of actions.

if (condition) {
    statement_1;
}
else {
    statement_2;
}

You can also check for more than just one condition by using the built in 'else if' statement. This can be done the following way:

if (condition_1) {
    statement_1;
}
else if (condition_2) {
    statement_2;
}
else {
    statement_3;
}

For Loops

Loops are ways of being able to repeat a specific line of code a set amount of times. With for loops, you can define how many times you would like the code to run. For example:

for (int i = 0; i < 3; i++) {
    statement;
}

Going through this bit by bit, let's see what this does

  • First you declare an integer i, and assign it the value 0
  • Then you check to see if the value i is less than 3. If so, continue running the code, if not, then stop
  • You run the code, and after doing so, add 1 (from i++) to i
  • Rinse and repeat

While Loops

While loops are also a form of loop, however, instead of incrementing a value a set amount of times, this loop will continue to run while a specific condition is true. Taking a look at this we have:

while (condition == true) {
    statement;
}

Notice we use '=='. This operator is used to check if the value on the left is equal to the value of the right. If so, it outputs true. If not, it outputs false.

Switch Statements

In some circumstances we need to check many different cases for a variable. While if else statements are great for this, if you are checking more than a few possible outcomes, the if else statement can quickly become too complex. Hence we have something called a switch statement.

It works as follows:

Say we define a variable like so

int sum = get_int("Enter a number between 1 and 5\n");

This promts the user to enter their age (As given by the <cs50> library).

Now we want to check the value that the user has entered and run some code based on the value given. As there are many possible values to check, we can use the switch statement like so

switch (num) {
case 1:
    statement_1;
    break;
case 2:
    statement_2;
    break;
case 3:
    statement_3;
    break;
case 4:
    statement_4;
    break;
case 5:
    statement_5;
    break;
}

Breaking this down we see the following:

  • First we say we are going to perform a switch statement
  • From there we then check each case, from 1 to 5
  • Depending on the value entered by the user, we will then execute the code we wish to use
  • Optionally, we then say break; as a way of ending the switch statement, so as not to waste resources checking the remaining statements
